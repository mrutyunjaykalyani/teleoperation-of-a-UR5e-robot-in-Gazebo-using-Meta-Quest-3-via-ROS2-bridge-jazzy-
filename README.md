# teleoperation-of-a-UR5e-robot-in-Gazebo-using-Meta-Quest-3-via-ROS2-bridge-jazzy-
This repository provides a high-performance bridge between the Meta Quest 3 and a UR5e robotic arm. By leveraging ALVR for wireless streaming and OpenVR for spatial tracking, this project enables 6-DOF (Degree of Freedom) real-time teleoperation of a robot in Gazebo or physical hardware using MoveIt Servo.
